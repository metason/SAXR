ecoANIM.json

Description

The definition of a sequence in config.json does specify seperate data scenes with anual data reps corresponding to the year of the time series stored in perCapita.json. datarepgen.pygenerates all anual scenes with their data reps and stores the result in datareps.json. 

    "sequence": {
        "field": "year",
        "domain": [
            2000,
            2024
        ]
    },

In the ARchi app the ecoANIM.json script does increase the data.idxcounter each second between 0 and 23. With the setVal()command the counter is applied to select the current data scene of the data visualization item. In parallel the year is shown in a label that is billboarding.
