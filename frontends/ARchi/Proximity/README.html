irisLOD.json

Description

By changing the definition of settings.json three different representations of the iris.json dataset has been generated:

  1. a non-transparent cluster plot
  2. a transparent cluster plot combined with a scatter plot
  3. a scatter plot

The three generated data viz representations including all generated panels have been manually merged into a list and stored in datareps.json.

In the ARchi app the irisLOD.json script does constantly evaluating the proximity of the user to the data vizualisation. Depending on the distance the active data representation is selected by setting the index of the data visualization with the setVal() function.
